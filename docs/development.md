# Development Process

1. Set up the project folder in VS Code.
2. Created a new Git repository using `git init`.
3. Wrote the core logic:
   - Random number generation
   - User input loop
   - Hint system
4. Added attempt counter.
5. Tested the program in VS Code terminal.
6. Added documentation and screenshots.
7. Created commits and pushed everything to GitHub.

The project follows a simple folder structure with clear documentation and clean code.
